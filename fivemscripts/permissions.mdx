You Can Setup GLOBAL permissions in the bridge. This means Access to ALL of the scripts

## Step 1: Navigate To Bridge Resource

Go To pimzu-bridge folder:

> /pimzu-bridge

Inside, you should see:

<Frame>
  ![Pb](/images/pb.png)
</Frame>

## Step 2: Open Config

> /pimzu-bridge/config.lua

```lua config.lua
Config.Override = {
	'license1',
	'license2'
}
```
